
# 🐍 Fundamentos de Python

## 📌 Descripción del proyecto

Este repositorio contiene el desarrollo de los laboratorios y ejercicios correspondientes a la guía de **Fundamentos de Python**, abarcando los temas de:

* Uso de la función `print()`
* Literales y manejo de cadenas
* Operadores matemáticos
* Variables y expresiones

El objetivo principal es demostrar la comprensión y aplicación práctica de los conceptos básicos del lenguaje Python mediante la resolución de problemas y ejercicios estructurados.

---

## 🎯 Objetivos de aprendizaje

* Comprender el funcionamiento de la función `print()`.
* Aplicar correctamente literales y cadenas.
* Utilizar operadores aritméticos respetando su jerarquía.
* Implementar variables en programas básicos.
* Resolver problemas mediante lógica de programación.
* Documentar soluciones de forma clara y estructurada.

## ⚙️ Requisitos

Para ejecutar este proyecto necesitas:

* Python 3 instalado
* Editor de código (recomendado: Visual Studio Code)
* Git instalado
* Cuenta en GitHub

---

## ▶️ Ejecución del proyecto

1. Clonar el repositorio:

```
git clone https://github.com/KelinMontoya/fundamentos_python.git
```

2. Acceder a la carpeta:

```
cd fundamentos_python
```

3. Ejecutar cualquier archivo Python:

```
python nombre_archivo.py
```
